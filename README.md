个人技术博客项目，用来沉淀日常开发记录、技术文章、专题笔记以及个人创作相关内容。

当前项目在默认文档能力的基础上，做了更贴近个人主页的首页改造，包括：

- 个人主页首页风格化布局
- 基于 Markdown 文章元数据的专题归档
- 首页博客 / 关于我 双标签切换
- 本地全文搜索
- 同专题文章的上一篇 / 下一篇引导
- 支持通过 Markdown 持续扩展博客内容
